## ⚙️ Як запустити локально

Для роботи скриптів рекомендується використовувати Python 3.10+. 

1. **Клонувати репозиторій:**
   ```bash

   cd KPI-DM-Labs
   ```
2. **Запуск конкретної роботи (наприклад, ЛР3):**
   ```bash 
   cd LR3
   ```

## ⚖️ Академічна доброчесність

Дані матеріали призначені для **ознайомлення** з принципами реалізації дискретних алгоритмів. 

⚠️ **Нагадування:** Копіювання коду без розуміння його роботи для здачі викладачу є порушенням [Кодексу честі КПІ](https://kpi.ua/academic-integrity). 

**Автор не несе відповідальності** за наслідки використання цих матеріалів третіми особами (ануляція балів, догани тощо), у разі якщо таке використання порушує пункти Кодексу честі КПІ.

**Рекомендації:**
- Використовуйте ці приклади для перевірки власних розрахунків.
- Звертайте увагу на складність алгоритмів (Big O) та структури даних (`dict`, `set`, `list`), що використовуються в реалізаціях. 🧠
